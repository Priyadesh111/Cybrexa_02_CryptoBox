
# CryptoBox - Password Strength Checker

## Description

CryptoBox is a password security tool built using HTML, CSS, JavaScript, and Regular Expressions (RegEx).

The application helps users evaluate password strength, detect weak password patterns, generate secure passwords, and improve overall password security through real-time analysis and recommendations.

---

## Features

### Password Strength Checker
- Real-time password analysis
- Weak, Medium, and Strong strength detection
- Strength meter visualization
- Uppercase letter validation
- Lowercase letter validation
- Number validation
- Symbol validation
- Password length validation (8-32 characters)

### Security Recommendations
- Detects missing password requirements
- Provides warnings and improvement suggestions
- Security awareness tips for creating stronger passwords

### Common Password Detection
- Checks passwords against a list of common and weak passwords
- Displays SAFE or WARNING status

### Password Generator
- Adjustable password length (8-32 characters)
- Uppercase letters option
- Lowercase letters option
- Numbers option
- Symbols option
- Random strong password generation

### Additional Features
- Suggest Strong Password button
- Copy Password button
- Browser-only password analysis
- Responsive design

## How to Run

1. Download or clone the project.
2. Open index.html in any web browser.
3. Enter a password to analyze its strength.
4. Generate secure passwords using the Password Generator.
5. Copy generated passwords using the Copy button.

---

## Security Note

Passwords are analyzed locally within the browser and are not stored, shared, or transmitted anywhere.
